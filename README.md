## Installation

Go to project directory and run (make sure you have node installed first)

```bash
  npm install
```

or

```bash
  yarn install
```

## Start localhost

```bash
  npm run dev
```

or

```bash
  yarn run dev
```

Click start: http://localhost:3000/

## Core Libraries Used

- [Next JS v13.1.6](https://reactjs.org/)
